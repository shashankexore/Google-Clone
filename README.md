# Google Clone

This project is a simple Google homepage clone built using HTML and CSS. The design mimics the layout and elements of Google's main page, such as the header with links and icons, the Google logo, a search bar, and buttons.

## Features

- **Header Section:**
  - Links for "About" and "Store" on the left.
  - Links for "Gmail", "Images", "Apps", and "Account" icons on the right.

- **Main Body Section:**
  - Google logo displayed at the top.
  - A search bar with icons for microphone and search.
  - Buttons for "Google Search" and "I'm Feeling Lucky".

## Technologies Used

- **HTML5:** Structure and content for the webpage.
- **CSS:** Styling the layout, positioning, and elements using Flexbox.
- **Material Icons:** Icons for search, microphone, and user account.

## Installation

1. Clone or download the repository.
2. Open the `index.html` file in your web browser to view the project.
